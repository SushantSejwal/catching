Stuff to do

open Microsoft Store and install files app.

why install files app? I didn't know which version of windows you are using and files app work on both windows 10 and 11.

Settings to do

Now enable 2 option in files app. 1st is the turning on Show hidden Files and Folders and show dot files and 2nd is turning on Show extensions for known file types.

these option might be enable. if yes, then make sure these options are enabled. click below to enabled the options

enable options

Enabling option

Follow these steps to enable options

Open Files App
Open Settings
Go to preferences
Find the option Files and Folders and expand it
enable the option show extension for known file type and unexpand the Files and Folder option
Find the option Hidden Files and Folders and expand it
enable show hidden files and folders AND show dot files

Searching for Photo , Video and Audio

we will serch for these in the Files App.

You will see 6 folders in the Home section of File Apps. Desktop, Documents, Downloads, Music, Picture and Video. the thing you have do is, open each folder one by one and look inside them one by one.

Usally the Desktop folder contains the item that are displayed on desktop, like trash can, and as the name suggest Documents, Downloads, Music, Pictures and Videos folder are used to store Documents, Downloads, Music, Pictures and Videos respectively.

but this doesn't mean videos will only be stored in videos folder. any file can be store in any folder.

Don't enter the Local Disk(C:) folder, that's for storing Operating System file. If you enter in it that's ok but please don't delete any file, it might cause any app to fail or your whole system might crash.
